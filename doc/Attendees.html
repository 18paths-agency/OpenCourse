    people() click to toggle source

    API interface for people that have a relationship to entities that allow for 'attendance'.

    Example¶ ↑

    course = Course.create
course.users << [User.first, User.second, User.third]
course.people # => [user1, user2, user3]

    As opposed to my original API: 

    course.people(course) # => [user1, user2]

    # File app/models/concerns/attendees.rb, line 18
def people
  self.users if respond_to? :users
end
